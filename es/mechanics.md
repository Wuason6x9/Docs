# Mechanics

**¿Qué es Mechanics?**  
Mechanics es un núcleo que contiene muchas bibliotecas y APIs que cada mecánica que crees puede usar este NÚCLEO.

**¿Qué es una mecánica?**  
Una mecánica es como un mod/plugin pero simplemente utiliza el núcleo de Mechanics.
